# Varatek Construction Company (Next.js App Router)

This site was migrated to Next.js while preserving the original UI 1:1.

## Getting started

1. npm install
2. npm run dev
3. Open http://localhost:3000

## Structure

- app/
  - layout.tsx
  - page.tsx
  - ClientBehavior.tsx
  - globals.css
- public/
  - images/

Global styles are in `app/globals.css`.
